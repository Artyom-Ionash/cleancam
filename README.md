# CleanCam 🎥

Минималистичное Windows-приложение для вывода изображения с веб-камеры в безрамочном окне поверх всех остальных окон.

## Особенности

- **Frameless:** Полное отсутствие системных рамок.
- **Always on Top:** Окно закреплено поверх других приложений.
- **Smart Resize:** Изменение размера мышкой с автоматическим сохранением пропорций кадра.
- **Tray Control:** Управление через системный трей (скрыть/показать/выход).
- **Zero-Config:** Автоматическое определение камеры и параметров.

## Архитектура UI (Atomic Design)

Интерфейс разделен на слои внутри директории `src/ui/`:

- `atoms/` — базовые неделимые компоненты (`HoverButton`, `VideoLabel`, `AppTrayIcon`).
- `molecules/` — группы компонентов со своей логикой верстки (`CameraControls`).

_Правило проекта: Фреймворк PyQt6 изолирован строго внутри папки `ui/`._

## Управление проектом (uv)

Проект использует [uv](https://github.com/astral-sh/uv) для управления зависимостями.

**1. Подготовка и запуск:**

```powershell
uv sync
uv run src/main.py
```

**2. Линтинг и проверка типов:**

```powershell
uv run pyright --watch
uv run ruff check . --fix
```

**3. Сборка в .exe:**

```powershell
uv run python src/scripts/compile.py
```

## Управление в приложении

- **LMB (в центре):** Перетаскивание окна.
- **LMB (за края):** Пропорциональное изменение размера.
- **Наведение курсора:** Появление панели инструментов (Смена камеры, Поворот, Закрыть).
- **Трей:** Правый клик по иконке камеры для вызова контекстного меню.
