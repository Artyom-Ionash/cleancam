# CleanCam 🎥

Минималистичное Windows-приложение для вывода изображения с веб-камеры в безрамочном окне поверх всех остальных окон.

## Особенности

- **Frameless:** Полное отсутствие системных рамок.
- **Always on Top:** Окно закреплено поверх других приложений.
- **Smart Resize:** Изменение размера мышкой с автоматическим сохранением пропорций кадра.
- **Tray Control:** Управление через системный трей (скрыть/показать/выход).
- **Zero-Config:** Автоматическое определение камеры и параметров.

## Разработка и сборка

Проект управляется диспетчером пакетов [uv](https://github.com/astral-sh/uv). Это гарантирует детерминированность сборки и изоляцию зависимостей.

### 1. Подготовка окружения

Убедитесь, что у вас установлен `uv`. Затем синхронизируйте зависимости:

```powershell
uv sync
```

_Это создаст виртуальное окружение `.venv` и установит `PyQt6`, `OpenCV` и `PyInstaller`._

### 2. Запуск в режиме разработки

```powershell
uv run src/main.py
```

### 3. Компиляция в .exe

Для создания портативного исполнимого файла используется кастомный скрипт сборки, который оптимизирует размер и очищает временные артефакты.

**Выполните команду:**

```powershell
uv run python src/scripts/compile.py
```

**Что происходит при сборке:**

- Используется режим `--onefile` (все зависимости внутри одного EXE).
- Исключаются неиспользуемые тяжелые модули `QtNetwork`, `QtWebEngine` и др.
- Временные файлы (`build/`, `*.spec`) направляются в изолированные папки.
- Готовый файл появляется в папке `dist/CleanCam.exe`.

### 4. Разработка:

Активация статического анализатора кода:

```powershell
uv run pyright --watch
```

## Технические детали сборки (для инженера)

- **Размер файла:** ~100 МБ. Основной вес составляют динамические библиотеки OpenCV и рантайм PyQt6.
- **Архитектура:** Сборка настроена через `src/scripts/compile.py`. Точка входа в `pyproject.toml` позволяет запускать процесс через `uv run`.
- **Кириллица в путях:** Проект корректно обрабатывает пути с кириллицей благодаря использованию `pathlib` и переходу на структуру `src/`.

## Управление

- **LMB (в центре):** Перетаскивание окна.
- **LMB (за края):** Пропорциональное изменение размера.
- **Наведение курсора:** Появление кнопок «Закрыть» и «Поворот 90°».
- **Трей:** Правый клик по иконке камеры для вызова меню.
